전체 안정화 수정 내용

1. /api/hospitals/{id} 상세 API 500 오류 수정
- DB에 web_hospital_reviews 테이블이 없는 경우에도 오류가 나지 않도록 처리했습니다.
- web_lcl_anomaly_reviews, web_lcl_anomaly_review_fallback 기준으로 관리도 리뷰를 우선 표시합니다.

2. h1 같은 더미 ID 내부 변환 오류 수정
- FastAPI Query 객체가 내부 함수 호출 때 들어와도 오류가 나지 않도록 수정했습니다.

3. CORS 설정 유지
- localhost:5173 프론트에서 백엔드 API 호출 가능하도록 설정했습니다.

4. 개선 인사이트 로직 유지
- 집중개선 키워드, 낮은 성과 카테고리, 강점 키워드 순으로 추천 액션을 생성합니다.

5. 차트 컨테이너 최소 크기 보강
- Recharts width/height 경고를 줄이도록 컨테이너 최소 크기를 추가했습니다.

실행 전 기존 백엔드는 Ctrl + C로 반드시 끄고 새 폴더에서 다시 실행하세요.
