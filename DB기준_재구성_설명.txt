DB 기준으로 코드를 다시 구성한 버전입니다.

핵심 변경
1. 백엔드가 SQLite DB를 기준으로 병원 상세 JSON을 완성해서 반환합니다.
2. 프론트는 중간 변환을 최소화하고 백엔드 JSON을 그대로 화면에 표시합니다.
3. 기존 mock 데이터 의존을 제거했습니다.
4. 병원 상세 화면에 필요한 항목은 모두 포함됩니다.
   - 종합 점수
   - 지역 내 순위
   - 수원시 전체 순위
   - 병원 전체 순위 리스트
   - 카테고리 레이더 차트
   - IPA 키워드 분석
   - 포지셔닝 맵
   - 관리도
   - LCL/이상 구간 리뷰
   - 맞춤형 개선 인사이트
   - 리포트 다운로드

실행 방법

1. 백엔드 실행
cd backend
py -3.11 -m pip install -r requirements.txt
py -3.11 -m uvicorn main:app --reload

2. 프론트 실행
프로젝트 최상위 폴더에서
npm install
npm run dev

3. 접속
http://localhost:5173

4. API 확인
http://127.0.0.1:8000/docs
http://127.0.0.1:8000/api/hospitals
