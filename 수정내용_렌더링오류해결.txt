수정 내용

1. React 오류 Objects are not valid as a React child 문제를 해결했습니다.
2. keyword 객체가 화면에 그대로 출력되지 않도록 safeText 유틸을 추가했습니다.
3. 관리도 리뷰 키워드 표시 부분에서 { keyword } 객체가 와도 문자열로 변환됩니다.
4. 추천 개선 액션과 키워드 분석 패널도 객체 데이터에 대응하도록 수정했습니다.
5. Recharts width(-1) 경고는 화면 컨테이너 크기 관련 경고라 치명 오류는 아니며 화면이 사라지는 주 원인은 React 객체 렌더링 오류였습니다.
