수정 내용

1. 수원시 전체 순위와 지역 내 순위가 0 또는 #1로 잘못 보이던 문제를 수정했습니다.
- 프론트에서 이미 정규화된 ranking 데이터를 다시 덮어쓰던 부분을 제거했습니다.
- cityRank, cityTotal, districtRank, districtTotal을 유지하도록 수정했습니다.

2. 맞춤형 개선 인사이트 출력 보강
- keywordPositions 기반으로 집중 개선 키워드 TOP5를 자동 생성합니다.
- keywordPositions 기반으로 강점 키워드를 자동 생성합니다.
- 카테고리 데이터가 비어도 화면이 깨지지 않도록 방어 처리했습니다.

3. 관리도 리뷰 표시 보강
- LCL 이상 지점이 있으면 첫 이상 지점을 자동 선택합니다.
- 점을 직접 클릭해도 리뷰 패널이 열립니다.
- 정확한 기간 매칭이 안 되면 fallback 리뷰를 표시합니다.

4. 병원 순위는 TOP20 제한을 제거하고 전체 순위로 표시합니다.

실행 방법

백엔드:
cd backend
py -3.11 -m uvicorn main:app --reload

프론트:
프로젝트 최상위 폴더에서
npm install
npm run dev
